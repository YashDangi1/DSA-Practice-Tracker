# DSA Practice Tracker (C++)

A console-based C++ application to track and analyze Data Structures & Algorithms (DSA) practice efficiently.  
This project helps users record problems they practice, analyze topic-wise progress, and identify weak areas.

- Add DSA problems with:
  - Problem name
  - Platform (LeetCode, CodeChef, etc.)
  - Topic
  - Difficulty level
  - Solved / Unsolved status
- Mark problems as **IMPORTANT**
- View all saved problems
- Search problems (case-insensitive)
- Delete problems
- View all important problems in one click
- Topic-wise analytics with solved percentage
- Overall solved vs unsolved statistics
